## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production  
npm run build
```

- Large: 1024px+ (lg:)
- XL: 1280px+ (xl:)

- Primary: Cyan (#22D3EE)
- Background: Dark gray (#111827)
