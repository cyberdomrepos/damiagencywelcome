Short intro: DamiAgency is a landing site built with Next.js (app router), TypeScript and Tailwind CSS.

Quick start

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

3. Build:

   npm run build

4. Start production preview:

   npm run start
